## About <a name = "about"></a>

This mini project is to demonstrate the bug in the github issue https://github.com/nodkz/mongodb-memory-server/issues/556

### Prerequisites

You need to have [MongoDB](https://www.mongodb.com/) installed and running. You also need to install [Node.js](https://nodejs.org/) on your machine.

### Installing

- Clone the repo
- Navigate to the root directory of the project
- Run `npm install`

## Usage <a name = "usage"></a>

To execute the code, navigate to the root directory of the project and run `npm run dev` in your terminal.
To run tests, run `npm test` in your terminal.
